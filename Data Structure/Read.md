# File Descriptions

This repository contains various C programs and data structure implementations. Below is a table of the files with links and brief descriptions.

| File Name                                      | Description                                                        |
|------------------------------------------------|--------------------------------------------------------------------|
| [PolynomialArray.c](PolynomialArray.c)         | Polynomial representaion using array solely.  |
| [PolynomialStruct.c](PolynomialStruct.c)       | Polynomial representation using array of structures.  |
| [SparseMatrix.c](Data%20Structure/SparseMatrix.c) | Implements operations for sparse matrix representations.         |
| [binarysearch.c](binarysearch.c)               | Contains the implementation of the binary search algorithm.       |
| [insertionsort.c](Data%20Structure/insertionsort.c) | Implements the insertion sort algorithm.                        |
| [linearsearch.c](Data%20Structure/linearsearch.c) | Implements the linear search algorithm.                          |
| [selectionsort.c](Data%20Structure/selectionsort.c) | Implements the selection sort algorithm.                         |
| [stack.c](stack.c)                            | Provides an implementation of a stack data structure.            |
